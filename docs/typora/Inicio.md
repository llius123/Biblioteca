# Typora apuntes

## Como tener los estilos que tengo en el typora del curro

Hay que cargarse lo siguiente:
blockquote --> eliminar font-style: italic;

#write --> 
#write {
	max-width: 40em;
}

@media only screen and (min-width: 1400px) {
	#write {
			max-width: 914px;
	}
}
